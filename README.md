# Modern Glassmorphism Portfolio

A professional, high-end portfolio website for students and early-career professionals. Built with a focus on minimalism, aesthetics, and smooth user experience.

## ✨ Features
- **Modern Aesthetic**: Dark mode, glassmorphic cards, and layered gradient backgrounds.
- **Responsive Design**: Optimized for all screen sizes (Mobile, Tablet, Desktop).
- **Interactive Elements**:
  - Hero typing animation.
  - Smooth scroll navigation.
  - Project filtering system.
  - Dynamic theme toggle (Dark/Light).
  - AI Assistant Chatbot (Digital Twin).
- **Sticky Navbar**: Background blur on scroll.
- **Glassmorphism**: High-end blur effects with subtle borders.

## 🛠️ Tech Stack
- **HTML5**
- **Tailwind CSS 4** (via Vite)
- **Vanilla JavaScript**
- **Vite** (Build Tool)

## 📦 Project Structure
- `index.html`: Main structure and content.
- `style.css`: Custom styles and Tailwind configurations.
- `script.js`: Interactivity and logic.

## 🚀 How to Run Locally

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Start Development Server**:
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) in your browser.

3. **Build for Production**:
   ```bash
   npm run build
   ```
   The production-ready files will be in the `dist/` directory.

## 🌐 Deployment
You can easily deploy this to **Netlify** or **Vercel**:
1. Connect your GitHub repository.
2. Set the build command to `npm run build`.
3. Set the publish directory to `dist`.

## 💡 Customization
- **Change Name/Role**: Edit the `<h1>` tag in `index.html`.
- **Update Projects**: Add or modify items in the `projects-grid` div in `index.html`.
- **Change Colors**: Modify the `--accent-primary` and background variables in `style.css`.
